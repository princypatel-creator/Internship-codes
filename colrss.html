Mastering CSS Margin

Margin creates empty space outside an element's border to separate it from others.

1. Individual Sides

Using margin-top, margin-bottom, etc. explicitly to push specific sides.

2. Four-Value Shorthand

Clockwise order: Top, Right, Bottom, Left (TRBL). Look at the asymmetric spacing around me.

3. Two-Value Shorthand

First value controls Top/Bottom, second controls Left/Right.

4. Uniform Margin

One single value applies equal spacing to all four sides simultaneously.

5. Horizontal Centering (margin: auto)

By giving a block element a fixed width and setting margin: 0 auto;, the browser balances the left and right space perfectly to center it.

6. Modern Logical Properties

Uses margin-block (Top/Bottom) and margin-inline (Left/Right). The modern standard for multilingual sites.
